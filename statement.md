# Project Statement -- Assignment Submission Reminder

## 1. Problem Statement

Students often struggle to keep track of multiple assignment deadlines,
leading to last‑minute stress or missed submissions. There is a need for
a simple, accessible, and lightweight tool that allows users to store
assignment details, track deadlines, and receive alerts for upcoming due
dates within the next 24 hours.

## 2. Scope of the Project

This project provides a basic command‑line based assignment reminder
system developed in Python.\
The scope includes: - Adding assignments with title, date, and time. -
Storing all assignments in a structured list. - Displaying all
assignments in chronological order. - Deleting assignments as needed. -
Checking for submission deadlines within the next 24 hours.

The project focuses on fundamental data handling using Python and is
ideal for beginners learning programming, date-time operations, and
logical structuring.

## 3. Target Users

-   College and school students who need a simple tool to track
    assignments.
-   Beginners learning Python and wanting a practical mini‑project.
-   Anyone who prefers a lightweight, offline deadline tracker without
    complex interfaces.

## 4. High-Level Features

-   **Add Assignment:** Store assignment title, submission date, and
    time.
-   **View All Assignments:** Sorted list of all stored assignments.
-   **Delete Assignment:** Remove unwanted or completed assignments.
-   **Upcoming Deadline Checker:** Identifies assignments due within the
    next 24 hours.
-   **Interactive Menu:** Simple CLI‑based navigation for ease of use.
